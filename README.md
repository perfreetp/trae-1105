# 导购助手 App

一个为服装店导购设计的移动端应用，帮助导购在店内接待顾客、推荐搭配和跟进会员。

## 功能特性

### 7个主要页面

1. **今日任务** - 查看待办任务、今日业绩概览、快捷操作入口
2. **顾客档案** - 会员列表搜索、会员详情查看、积分优惠券查询
3. **商品查询** - 扫码查款、按类别/风格/颜色筛选、库存查询
4. **搭配推荐** - 搭配方案展示、创建新搭配、拍照保存搭配
5. **试衣记录** - 试衣清单管理、状态更新、成交/流失原因记录
6. **会员跟进** - 跟进任务列表、回访结果填写、批量关怀操作
7. **业绩统计** - 销售目标进度、核心指标、周业绩趋势、排名

### 核心功能

- ✅ 扫码查尺码库存
- ✅ 按颜色和风格筛选商品
- ✅ 查看同款门店余量
- ✅ 记录顾客偏好
- ✅ 创建试衣清单
- ✅ 拍照保存搭配
- ✅ 推荐关联单品
- ✅ 预约到店提醒
- ✅ 发送会员关怀
- ✅ 登记成交或流失原因
- ✅ 申请调货
- ✅ 查看个人销售目标
- ✅ 领取店长任务
- ✅ 填写回访结果
- ✅ 查询积分和优惠券
- ✅ 生成接待小结

## 技术栈

- React 18 + TypeScript
- Vite 构建工具
- Vant UI 移动端组件库
- React Router 路由管理

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```
